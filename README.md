# FUTURE_FS_02
Full Stack Web Development Internship - Task 2 (FS Track). Mini CRM application built using modern web technologies with CRUD functionality.
It is designed to manage customer information efficiently using full-stack development principles.

## Features
- Add new customers
- View customer list
- Update customer details
- Delete customer records
- RESTful API integration

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Version Control: Git & GitHub
